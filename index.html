Qual é a importancia de um bom Web design?

Um bom é muito importante para a organizaçâo de uma pagina, facilitando o uso para todos os publicos

Como fazer um bom web design?

Para fazer um bom web design seu site precisa ser bem organizado, além de acessivel para todas as plataformas

Como alcançar todas as plataformas?

Para alcançar todas as plataformas é necessário que sua pagina se adapte a todos os tamanhos de tela, para isso é necessário usar o media quarie, o flex-box e etc.
